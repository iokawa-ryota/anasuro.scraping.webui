# 起動方法ガイド

スロット店舗スクレイピング Web UI の起動方法は3つあります。

## 方法 1: start.bat（推奨）

```
start.bat をダブルクリック
```

**特徴：**
- ✅ シンプルで分かりやすい
- ✅ サーバーは最小化ウィンドウで起動
- ✅ ブラウザが自動で開く
- ✅ このウィンドウは5秒後に自動で閉じる
- ⚠️ サーバーはバックグラウンドで動作（見えない）

**サーバーを停止する場合：**

1. **タスクマネージャーを開く**（Ctrl+Shift+Esc）
2. **python.exe を探す**
3. **右クリック → タスクの終了**

または、タスクバーから Flask Server ウィンドウを探す。

---

## 方法 2: start_silent.bat（完全非表示版）

```
start_silent.bat をダブルクリック
```

**特徴：**
- ✅ サーバーが完全に隠れている
- ✅ ウィンドウが一切表示されない
- ✅ ブラウザだけが開く
- ✅ 最もスッキリした見た目
- ⚠️ サーバーの状態が分かりにくい

**サーバーを停止する場合：**

1. **タスクマネージャーを開く**（Ctrl+Shift+Esc）
2. **python.exe を探す**
3. **右クリック → タスクの終了**

---

## 方法 3: コマンドラインから実行（カスタマイズ向け）

```bash
python app.py
```

**特徴：**
- ✅ サーバーの動作が見える（デバッグに便利）
- ✅ エラーメッセージがリアルタイムで表示
- ✅ ログが見やすい
- ⚠️ ウィンドウが常に表示される

**ブラウザを開く：**

1. 別のコマンドプロンプトで実行、または
2. 手動でブラウザを開いて `http://localhost:5000` にアクセス

**サーバーを停止する場合：**

- Ctrl+C を押す

---

## 推奨される使い分け

| シーン | おすすめ |
|--------|---------|
| 通常の使用 | **start.bat** |
| 見た目を気にしない | **start.bat** |
| ウィンドウを一切見たくない | **start_silent.bat** |
| 開発・デバッグ中 | **コマンドライン実行** |

## よくある質問

### Q: サーバーが起動しているか確認したい

**A:** ブラウザで `http://localhost:5000` にアクセスして、ページが表示されるか確認してください。

### Q: ウィンドウを復元したい（start.bat の場合）

**A:** タスクバーで最小化されたウィンドウをクリックするか、タスクマネージャーから確認できます。

### Q: サーバーが停止しない

**A:** 以下の手順で強制終了できます：
1. Ctrl+Shift+Esc でタスクマネージャーを開く
2. `python.exe` を検索して選択
3. 「タスクの終了」をクリック

### Q: ポート 5000 が既に使われていると言われた

**A:** 別のアプリケーションが使用しているか、前回のサーバーが完全に終了していません：

```bash
# ポート 5000 を使用しているプロセスを確認（PowerShell で実行）
netstat -ano | findstr :5000

# プロセス ID を確認して終了
taskkill /PID <PID> /F
```

または、別のポートで起動：

```python
# app.py の最後を編集
if __name__ == '__main__':
    app.run(debug=True, host='localhost', port=8080)
```

## トラブルシューティング

### ブラウザが開かない場合

```bash
# デフォルトブラウザが設定されていない可能性があります
# 手動で開く場合：
1. ブラウザを起動
2. アドレスバーに http://localhost:5000 を入力
```

### サーバーが起動していないように見える

```bash
# サーバーが起動しているか確認（PowerShell で実行）
netstat -ano | findstr :5000

# または curl で確認（Windows 10 以降）
curl http://localhost:5000
```

### Python 実行時エラーが出た場合

```bash
# セットアップを再確認
python setup_check.py
```
